# 小程序到 iOS 功能对照表

本表是首个全量版本的功能验收基线。所有领域均为同一交付范围，不按优先级拆分。

| 领域 | 现有能力 | iOS 目标 | 范围 |
|---|---|---|---|
| 身份 | 账号密码、微信绑定、教师/家长角色 | 手机验证码、Apple 登录、微信迁移绑定、多角色 | 全量首版 |
| 用户 | 注册、登录、资料、切换身份、退出 | 登录会话、资料、角色空间、账号删除 | 全量首版 |
| 学生 | 学生档案、绑定孩子、孩子切换 | 学生档案、监护人关系、孩子切换 | 全量首版 |
| 课程 | 课程模板创建、编辑、资料 | 课程模板与教学内容 | 全量首版 |
| 班级 | 创建、编辑、邀请码、加入、移除学生 | 完整班级生命周期与成员管理 | 全量首版 |
| 排课 | 自动生成、日历、手动课节、重新生成 | 周/月日历、自动与手动课节、调课 | 全量首版 |
| 上课 | 确认、取消、撤销、课后反馈 | 事务化确认、考勤、反馈、撤销 | 全量首版 |
| 课时 | 充值、消费、调整、余额、流水、预警 | 追加式课时账本、余额、流水、预警 | 全量首版 |
| 考勤 | 单次/批量考勤、统计 | 课节内考勤与统计 | 全量首版 |
| 首页 | 今日课节、快捷入口、统计、公告 | 角色化首页与增量刷新 | 全量首版 |
| 通知 | 站内信、微信订阅消息 | 站内信、APNs、微信兼容、偏好设置 | 全量首版 |
| 教学 | 作业、提交、批改、资料 | 作业闭环、资料上传下载 | 全量首版 |
| 学习 | 计划、打卡、错题本、积分 | 迁移现有可用能力 | 全量首版 |
| 账单 | 手工记录缴费、退款、统计 | 教学订单、支付记录、全额结算、协助退款 | 全量首版 |
| VIP | 激活码与到期时间 | StoreKit 订阅、服务端权益、内部赠送 | 全量首版 |
| 经营 | 收入、课时、班级统计 | 教师经营概览 | 全量首版 |
| 公告反馈 | 公告、问题反馈 | 公告、反馈和客服入口 | 全量首版 |

## 微信专属能力替换

| 微信能力 | iOS 替换 |
|---|---|
| `wx.cloud.callFunction` | `URLSession` 调用版本化 REST API |
| `OPENID` | `users.id` + `user_identities` |
| `wx.Storage` | Keychain + SwiftData/SQLite + UserDefaults |
| 微信订阅消息 | APNs + 站内通知，微信作为兼容通道 |
| `wx.cloud.uploadFile` | 预签名上传 URL + 对象存储 |
| 微信用户资料 | 用户主动填写 + Apple/手机号登录资料 |
| 自定义 TabBar | 原生 TabView，保留品牌视觉 |
