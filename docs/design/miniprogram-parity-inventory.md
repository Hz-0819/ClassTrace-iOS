# 微信小程序到 iOS 页面等价清单

视觉基线来自只读目录 `ClassTrace/miniprogram` 的 WXML、WXSS、页面配置和 72 个原始 PNG 素材。

## 一级导航

| 小程序入口 | iOS 入口 | 状态 |
| --- | --- | --- |
| 首页 | 首页 | 已改为蓝色装饰头部、提醒、快捷入口、课程卡片与经营概览 |
| 课程 | 课程 | 已改为今日安排、班级/学生/课程模板入口和小程序卡片结构 |
| 我的 | 我的 | 已改为资料头部、统计卡片、按教师/家长身份显示的功能分组 |
| 悬浮自定义 TabBar | 悬浮自定义 TabBar | 已使用原 `home/class/user` 普通和选中图标 |

## 完整功能入口

- 班级：创建、编辑、删除、邀请码加入、成员管理、班级详情。
- 学生：创建、编辑、删除、绑定孩子、监护关系、考勤统计。
- 课程与课节：课程模板、排课、调课、取消、确认上课、出勤、课后反馈、课时账本。
- 学习：作业、提交与批改、教学资料、学习计划和打卡、错题本。
- 账户：本地注册登录、账号设置、消息通知、公告、反馈、关于和管理员入口。
- 经营：经营概览、购课账单、外部收款记录、退款、VIP 权益。

## 已知验证边界

本地 Windows 环境不能运行 Xcode/iOS Simulator，因此本轮只能以原 WXML/WXSS 和素材做结构对照，并通过 GitHub macOS 工作流完成编译与单元测试。安装到 iPhone 后仍需按相同页面状态进行逐屏视觉对照。
